# Gestor de Tareas en Python

Aplicación de consola desarrollada en Python para gestionar tareas con operaciones CRUD, persistencia en JSON y filtros por estado y prioridad.

Proyecto realizado como práctica de refactorización, POO y organización modular del código.

## Funcionalidades

- Crear tareas
- Listar tareas
- Cambiar estado de una tarea
- Eliminar tareas
- Guardar tareas en archivo JSON
- Filtrar tareas por estado
- Mostrar tareas prioritarias
- Registrar fecha de creación

## Tecnologías utilizadas

- Python 3
- JSON
- Programación Orientada a Objetos (POO)
- Git / GitHub

## Cómo ejecutar

1. Clonar el repositorio:
```bash
git clone https://github.com/fpachilla/gestorTareas.git
```

2. Ingresar al proyecto
```bash
cd gestorTareas
```

3. Ejecutar
```bash
python main.py
```
